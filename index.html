Photogrammatron

Photogrammatron is an in browser photo editor that allows you to do some basic photoshop functions as well as apply some simple filters. I built this as a project to gain a better understanding of image manipulation and working with the HTML5 canvas element. It takes avantage of Facebook Oauth and the Facebook API so the user can edit a photo and then post that photo directly to their wall.


Craigslist Clone

I decided to build a Craigslist styled application. It utilizes the Flask framework alongside with SQLAlchemy, Flask-mail, and a mySQL database. It features handrolled CSRF protection and other security protocols to help keep the user protected.
